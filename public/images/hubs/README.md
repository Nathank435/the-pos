# Business-type hub photos

Drop a photo here named by the hub **slug** to replace the branded placeholder banner
on `/pos-systems/<slug>`:

`cafes.jpg`, `restaurants.jpg`, `retail.jpg`, `market-stalls.jpg`, `pubs-bars.jpg`,
`salons.jpg`, `tradesmen.jpg`, `takeaways.jpg`, `mobile-businesses.jpg`

Banners render at a 21:9 ratio — export ~1600×686px, optimised JPG/WebP. The
`<EditorialImage>` component applies the house treatment (slight desaturation, higher
contrast, warm gradient foot), so supply natural, documentary-style frames.

Recommended search strings live in `data/businessTypes.ts` (`imageHint`). See `BRAND.md`
for the full art direction.
